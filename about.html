About the Student and Project

Student Introduction

My name is Maahi Patel, and I created this website for IT 3203 Introduction to Web Development. The project combines research about HTTP with practical use of semantic HTML, external CSS, JavaScript, accessibility, and responsive design.

Project Goals

  • Explain HTTP with accurate, organized research
  • Create consistent navigation across all pages
  • Support desktop, tablet, and mobile screens
  • Build an interactive, accessible quiz

Technical Skills

  • Semantic HTML5 elements
  • CSS Grid and Flexbox
  • Media queries and fluid sizing
  • Responsive images and tables
  • JavaScript event handling and feedback

Responsive Strategy

The site uses a fluid content container, flexible grids, a collapsible mobile menu, touch-friendly controls, scalable typography, and horizontally scrollable tables on narrow displays.

Milestone Progress

Milestone 1 established the research site and shared visual design. Milestone 2 added the self-assessment quiz and corrected the footer layout. Milestone 3 finalizes the research and redesigns every page for responsive and mobile use.

This website is a class project for IT 3203 Introduction to Web Development at Kennesaw State University.
